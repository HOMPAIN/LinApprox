# LinApprox
Простая линейная аппроксимация для разного количество коэффициентов.

Позволяет выполнить линейную аппроксимацию для уванений вида:
`y = a*x1 + b*x2 + c*x3 + d*x4...`, где:

`a, b, c, d` - искомые коэффиценты,

`{y, x1, x2, x3, x4...}` - набор точек измерения.

## Пример для 4 коэффициентов
```cpp
LinApprox4 approx4;

// Добавляем точки (не менее 4)
approx4.Add(100,  10, 12,  22, 1);
approx4.Add(120,  30, 12,  62, 1);
approx4.Add(1000, 10, 1002, 22, 1);
approx4.Add(1200, 40, 1001, 72, 1);
approx4.Add(1250, 40, 1031, 72, 1);

// Расчёт коэффициентов
double a, b, c, d;
approx4.Calc(a, b, c, d);
```
